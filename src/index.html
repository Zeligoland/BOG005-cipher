Para ocultar mensaje

El mensaje se ocultará a través del método Cifrado César, para que tu mensaje se oculte correctamente, es necesario que nos indiques el número de desplazamientos que tendrá cada una de las letras.

Para descifrar mensaje

El mensaje se descifrará a través del método Cifrado César, para que podamos descifrarlo correctamente, la persona que te lo envió debe indicarte con qué offset está.
